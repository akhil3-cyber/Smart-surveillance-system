# Smart-surveillance-system

ASmart Surveillance System is an advanced security and monitoring solution that uses CCTV cameras, IoT devices, Artificial Intelligence (AI), and Machine Learning (ML) to monitor areas in real time, detect suspicious activities, and send instant alerts to authorized users. Unlike traditional surveillance systems that only record video, smart surveillance systems can automatically analyze video streams, identify objects or people, detect motion, recognize faces, and respond to security threats without continuous human supervision.
